# 🍽️ Knowledge Meals

> 지식을 맛있게 먹는 나만의 학습 노트 웹서비스

## 📖 프로젝트 소개

Knowledge Meals는 공부한 내용을 음식 메뉴처럼 기록하고 관리할 수 있는 React 기반 학습 노트 웹서비스입니다.

평소 공책에 필기하며 공부하는 습관에서 아이디어를 얻어 제작하였습니다. 손으로 작성한 노트는 분실 위험이 있고 원하는 내용을 찾기 어려운 문제가 있습니다. 이를 해결하기 위해 웹에서 과목별 학습 내용을 정리하고, 검색하며, 학습량까지 확인할 수 있도록 구현하였습니다.

---

## 🛠️ 기술 스택

### Front-End

* React
* JavaScript
* HTML5
* CSS3
* Vite

## ✨ 주요 기능

### 📝 지식 추가하기

* 과목 입력
* 제목 입력
* 학습 내용 입력
* 태그 입력
* LocalStorage 저장

### 🔍 지식 검색

* 과목 검색
* 제목 검색
* 태그 검색

### 📚 노트 목록

* 저장된 노트 조회
* 노트 삭제
* 최신 학습 내용 확인

### 📊 학습 분석

* 과목별 학습량 확인
* 학습 비율 시각화
* 가장 많이 학습한 과목 확인

### 🍽️ 오늘의 추천 메뉴

* 저장된 노트 중 하나를 추천 메뉴로 표시
